Course Description

The Object-Oriented Programming (OOP) course provides an in-depth introduction to the concepts of object-oriented design and programming. Students will learn how to design, develop, and analyze software using object-oriented techniques. Topics include classes, objects, inheritance, polymorphism, encapsulation, and abstraction. The course will also cover advanced topics such as exception handling, file I/O, and design patterns.
